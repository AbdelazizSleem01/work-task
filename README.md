#  لوحة تحكم المبيعات (Sales Dashboard)

---

## نظرة عامة على المشروع
لوحة تحكم **متكاملة وسريعة الاستجابة (Fully Responsive)** لإدارة المبيعات، تتيح:
- عرض إحصائيات فورية
- إدارة الطلبات (إضافة، تعديل، حذف)
- تغيير الثيم (فاتح/داكن)
- تبديل اللغة (عربي/إنجليزي)
- حفظ جميع الإعدادات تلقائيًا حتى بعد إعادة التحميل

---

## المميزات الرئيسية

| الميزة | الوصف |
|--------|-------|
| **واجهة مستخدم حديثة** | تصميم أنيق باستخدام CSS Flexbox + Grid + Variables |
| **فول ريسبونسيف** | يعمل بسلاسة على **الموبايل، التابلت، اللاب توب، والديسكتوب** |
| **دعم RTL/LTR** | تبديل تلقائي للاتجاه عند تغيير اللغة |
| **ثيم داكن/فاتح** | حفظ الثيم عبر `localStorage` |
| **50 طلب افتراضي** | بيانات حقيقية (أسماء مصرية + منتجات إلكترونية حقيقية) |
| **إدارة الطلبات** | إضافة، تعديل، حذف مع تأكيد |
| **جدول ذكي** | ترميز الألوان للحالة + زر تحرير/حذف |
| **مودال مرن** | نموذج إضافة/تعديل مع تحقق من الحقول |
| **لا قواعد بيانات خارجية** | كل شيء يُحفظ في `localStorage` |
| **أداء عالي** | لا إعادة تحميل، تحديث فوري عبر JS |
